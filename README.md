# Quantogral

Quantogral is a source-available quantitative trading framework for building, testing, and evaluating trading strategies written in Python and C++.

The goal is to provide a clean and extensible research environment where users can develop ideas, run backtests, analyze results, and later connect strategies to paper or live trading infrastructure.

Quantogral is currently in early development.
